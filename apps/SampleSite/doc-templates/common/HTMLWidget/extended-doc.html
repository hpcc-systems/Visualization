Attaches widget to an element on the page.

NOTE: The size() must be set before the target() and the target() must be set before calling render().


.target( element_id )

element_id
Type: String
A string representing the ID of the target element.

.target( element )

element
Type: Element
One DOM element to assign as the target container for this widget.

.target( SVGElement )

SVGElement
Type: SVGElement
One SVGElement to assign as the target container for this widget. If the widget being created is an HTMLWidget, a div will be appended to the page and positioned "over" the provided SVGElement.
