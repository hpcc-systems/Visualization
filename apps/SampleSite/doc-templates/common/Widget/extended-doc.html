Sets the dimensions of the widget.
.size( sObj )
sObj
Type: Object
An object with two properties, width and height. Ex: widget.size({ width: 100, height: 100 });
Enable/disable visibility of the widget.
Hiding the widget will prevent it from displaying and will visually preserve the space that the widget would have otherwise consumed.
An array of column names describing the data being input/displayed.
Render the widget immediately within the element passed to .target()
.render( callback )
callback
Type: Function
An optional callback function as parameter. The current widget object being operated on is passed to the function. The function will execute after the widget has completed rendering.
Example

widget.render(function(widget) { console.log(widget); });
