• Pop Up Examples

Dropdown

A typical dropdown, such as a dropdowm menu, where the content is displayed below the element calling it. Use the button to toggle the popup.

Dropleft/right

The same principle as a dropdown, except the content is displayed to the right or left of the element calling it.

Modeless Dialog

The Modeless dialog is a child window that allows the user to interact with the main page while the dialog is still on the screen.

Dropup

The same principle as a dropdown, except the content is displayed above the element calling it (useful at the bottom of the viewport).

Modal Dialog With Mask

The modal dialog is used to prevent any interaction with the parent page until the dialog is dismissed. Typically used with a background mask (but the opacity can be changed as needed).

Modal Dialog (Click Outside)

Sometime modals allow the user to simply click anywhere outside the modal to close it and return to the main page.
