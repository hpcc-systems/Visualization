Visualization Framework

  • Demo/Test Page and its source
  • Dermatology Test Page and its source
  • Wiki

The goal of the HPCC Visualisation Framework is to provide a comprehensive set of visualizations, adhering to a consistent set of interfaces. It includes wrappers for third party chart libraries (Google, C3 Charts, AM Charts etc.) as well as home grown visualizations. It also adds a set of connectors and marshallers to make connecting visualizations to the HPCC Platform a trivial excercise.

Standing on the back of giants

None of this would would be possible without the great work of others. The widgets/lib folder includes the work from:

  • requirejs
  • D3
  • Topojson
  • dagre
  • Font-Awesome
  • ColorBrewer
  • C3
  • AM Charts
  • d3-cloud

While widgets/src also includes wrappers for CDN hosted visualizations:

  • Google Maps
  • Google Charts
